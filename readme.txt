== Description ==
An experimental full site editing theme for blogs. 
This theme requires Gutenberg version 10.6 or newer. 
Clayi includes multiple block styles and block patterns including site headers, site footers and page layouts.

Name inspired from my little one name Clairine -> clay. but Clay is exist in theme WordPress.org so changed to Clayi :)
